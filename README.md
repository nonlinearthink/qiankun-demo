# Qiankun 微前端实战

本仓库为个人学习微前端搭建的案例。

## 主应用 main-app

使用的技术栈：`react`、`typescript`、`umijs`、`qiankun`

### 运行

```
yarn start
```

## 微应用 react-sub-app

使用的技术栈：`react`、`typescript`、`umijs`、`qiankun`

### 运行

```
yarn start
```

## 微应用 vue-sub-app

使用的技术栈：`vue3.0`、`typescript`

### 运行

```
npm run serve
```
